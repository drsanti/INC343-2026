# **INC343: Computer-Based Control and Monitoring Systems**

## Course Overview

This course introduces computer-based systems used for **control and monitoring of devices and machinery**. Students will gain both **theoretical knowledge** and **practical programming skills**, with an emphasis on real-world implementation.

The course is delivered using a **hybrid learning format**, combining **onsite** and **online** instruction. Each session includes **lectures** alongside **hands-on programming activities**.

## Course Description

This course covers the principles and applications of **computer-based control and monitoring systems**. Topics include the **hardware and software components** used in monitoring and control, **software design and development for automation systems**, and **industrial standard communication protocols**. The course also explores **Internet of Things (IoT) technologies**, **web-based software development** for control and monitoring applications, **computer and embedded system data communication**, as well as **data management, analysis, and visualization**. Emphasis is placed on the development of **Internet-based monitoring and control applications** for real-world industrial and automation environments.

---

## Learning Outcomes

Upon successful completion of this course, students will be able to:

1. **Explain** the operation and architecture of computer-based **control and monitoring systems**.
2. **Develop computer programs** to control devices and machinery using **standard industrial communication protocols**.
3. **Design and implement web-based applications** that enable **control and monitoring of devices and machinery** through a web browser.
